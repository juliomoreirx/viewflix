# 🎬 ViewFlix Bot

Streaming inteligente via Telegram. Pague apenas pelo que assistir!

## 🚀 Features

- 🎥 35.000+ filmes
- 📺 8.000+ séries
- 💰 Pagamento via PIX
- ⏰ Filmes: 24h | Séries: 7 dias
- 🔒 Proteção DRM
- 📱 100% pelo Telegram

## 🛠️ Tecnologias

- Node.js 18+
- MongoDB
- Telegram Bot API
- Mercado Pago API
- Express.js

npm install
cp .env.example .env
# Configure suas variáveis
